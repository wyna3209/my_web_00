# GitHub Pages에 프로젝트 올리기 (Windows 기준)

1. GitHub에 새 저장소(repository) 생성
   - https://github.com 에서 로그인 후, 새 저장소를 만든다.
   - 저장소 이름은 자유롭게 설정 (예: my_web_00)

2. 로컬 프로젝트 폴더에서 git 초기화
   - PowerShell에서 프로젝트 폴더로 이동
     cd d:\webs\my_web_00
   - git 초기화
     git init

3. 원격 저장소 연결
   - 원격 저장소 주소 복사 (예: https://github.com/사용자명/my_web_00.git)
   - 연결 명령어 실행
     git remote add origin https://github.com/사용자명/my_web_00.git

4. 파일 추가 및 커밋
   git add .
   git commit -m "초기 커밋"

5. main 브랜치로 푸시
   git branch -M main
   git push -u origin main

6. GitHub Pages 설정
   - GitHub 저장소 페이지로 이동
   - Settings > Pages 메뉴 클릭
   - "Branch"에서 main 선택, "/ (root)" 선택
   - 저장

7. 배포 주소 확인
   - 잠시 후 "Your site is published at ..." 링크가 생성됨
   - 해당 링크에서 웹사이트 확인 가능

# 참고
- index.html 파일이 반드시 있어야 함
- 변경사항이 있을 때마다 add/commit/push 반복
- style.css, script.js 등도 자동으로 반영됨
